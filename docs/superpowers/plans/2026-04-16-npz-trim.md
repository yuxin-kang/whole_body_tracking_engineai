# NPZ Trim Tool Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build an independent Isaac Sim trimming tool that replays a motion `.npz`, lets the operator mark logical start/end frames with keyboard shortcuts, and exports a trimmed `.npz` while preserving the original archive format.

**Architecture:** Keep interactive Isaac Sim control in `scripts/trim_npz.py` and move file-format-sensitive trim behavior into a pure utility module under `whole_body_tracking.utils`. This keeps the export logic unit-testable and limits Isaac Sim dependencies to the script entrypoint.

**Tech Stack:** Python 3.10, NumPy, pytest, Isaac Lab / Isaac Sim input APIs, existing `MotionLoader` replay path.
